## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd React
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage

Start the development server:

```bash
npm start
```

Open your browser and navigate to `http://localhost:3000`.
